## 📖 Project Overview

**Pet Hotel Management System** is a full-stack web application designed to manage 
day-to-day operations of a pet boarding hotel. The system supports room booking, 
pet check-in/check-out, daily care logging, and payment processing.

This project was developed as part of the CN230 Database Systems course, 
focusing on real-world database design and application development.
